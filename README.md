# New Year Resolution: Makeover Monday (Tableau Dashboard)

## üìä Project Overview

This project is a data visualization created as part of the **Makeover Monday** Week 3(2024) challenge. It focuses on analyzing data related to the popularity and distribution of various **New Year's Resolutions** across different demographic groups.

The dashboard aims to provide clear, comparative insights into resolution trends based on the respondents' gender and age.

* **Visualization Tool:** Tableau Public
* **Source File:** `New year resolution #MakeoverMonday.twbx` (Tableau Packaged Workbook)
* **Data Source:** The original dataset provided by the Makeover Monday challenge (details below).

## üí° Key Features of the Dashboard

* **Interactive Visualization:** Allows users to filter and explore data based on **gender** and different **age demographics**, as well as **resolution types**.
* **Design Focus:** A clean, dark chalkboard aesthetic utilizing bar charts to clearly compare resolution popularity across different gender and age groups.
* **Key Findings:**
    * **Health is a Universal Goal:** "Doing more exercise or improving fitness" is the highest-voted resolution for both male and female respondents.
    * **Priorities Shift with Age:** Younger adults (18-49) primarily focus on financial security, with "Saving more money" being a top resolution. Conversely, health and physical appearance dominate the priorities of the 50+ age groups.
    * **Gender Differences in Career Ambition:** The data highlights a notable difference, with males significantly more likely to pursue a career ambition compared to females.
